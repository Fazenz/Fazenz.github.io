SkyCityBuilders
Group: 1 person
Time: 35h
SkyCityBuilders is a small game that runs in the terminal. The game is simple and easy, built using iJava (A specific language for Lille University).
The game consits of a map, you can chose the size of it, you have to place specific colleting buildings: Steel mine, Gold mine, etc.
You place these buildings in the map as you want and in order to pass the levels you have to build the special building (Town Hall for the lvl 1).
